About GDG-Pune

The Place for Developers to Discuss Google Technologies Pune Google Developer Group meets on the once every month. Our members are using or want to learn more about developing applications using Google technology, including: - Google Ajax APIs - maps, search, visualization, gadgets (JavaScript), Android - mobile development (Java) - Google App Engine - scalable cloud computing (Python and Java) - Google Web Toolkit - web development platform (Java) - Project Hosting.
